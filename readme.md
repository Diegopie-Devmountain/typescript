# TypeScript

## Install
If you want to commit to using TS, it's better to install the compiler globally
`npm i -g typescript`

run the compiler using `tcs`, such as `tsc -v`
You can manually compile files by running `tsc path-to-file`

## inline type annotation

`let age: number = 10;`

`tsc --init`
